# DreamBridge AI — API Contract

Base URL: `http://localhost:8000/api`

Full interactive docs at `/docs` (Swagger, auto-generated by FastAPI).

| Method | Path | Purpose |
|---|---|---|
| POST | `/upload` | Upload a lesson file (PDF/image), creates a Lesson |
| POST | `/lessons/{id}/extract` | Extract text from the stored file (OCR / PDF parse) |
| POST | `/lessons/{id}/summarize` | Generate a summary of the extracted text |
| POST | `/lessons/{id}/quiz` | Generate a quiz from the lesson |
| POST | `/lessons/{id}/translate` | Translate lesson/summary into a target language |
| POST | `/lessons/{id}/personalize` | Generate a personalized version for a student profile |
| GET | `/lessons/{id}` | Get a lesson and all generated artifacts |
| POST | `/students` | Create/update a student profile |
| GET | `/students/{id}` | Get a student profile |
| POST | `/tutor/chat` | Chat with the AI tutor, grounded in a lesson |
| POST | `/progress` | Log a progress event (score, time spent) |
| GET | `/progress/{student_id}` | Get a student's progress history |
| GET | `/teacher/lessons` | List all lessons for a teacher (dashboard) |
| GET | `/teacher/students` | List all students for a teacher (dashboard) |

All request/response bodies are defined as Pydantic models in
`backend/app/models/schemas.py` and shown live in Swagger.
